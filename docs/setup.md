# Setup Guide

Follow these steps to set up the Civic Resolve platform on your local environment.

## Prerequisites

- **Python**: 3.9 or higher
- **Database**: MySQL Server (running locally)
- **Dependencies**: pip (Python package installer)

## Installation Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/wanderingCoder95/Civic-Resolve-Intelligent-Complaint-Routing.git
   cd Civic-Resolve-Intelligent-Complaint-Routing/project
   ```

2. **Install Dependencies**
   ```bash
   pip install flask pandas scikit-learn mysql-connector-python
   ```

3. **Configure Database**
   - Log into your MySQL instance.
   - Execute the initialization scripts found in `project/data/sql/` in the following order:
     1. `users.sql`
     2. `complaints.sql`
     3. `syslogs.sql`
   - Update the `dbconfig` dictionary in `project/app.py` with your local MySQL credentials.

## Running the Application

1. **Start the Flask Server**
   ```bash
   python app.py
   ```

2. **Access the Dashboard**
   - Open your browser and navigate to `http://localhost:5000`.
   - Use the sample credentials from `data/users.csv` to log in as a Student or Staff member.

## Troubleshooting

- **Database Connection Error**: Verify that your MySQL server is running and the credentials in `app.py` match your local setup.
- **Model Loading Issues**: Ensure that `model.pkl` and `vectorizer.pkl` are present in the project root. If missing, they can be generated by running the cells in `model.ipynb`.
